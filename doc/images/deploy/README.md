# 部署指南配图目录

该目录用于存放 [部署指南](../../部署指南.md) 中提到的 CloudBase 控制台截图。

建议优先补齐以下图片：

1. `cloudbase-env-overview.png`
   CloudBase 环境总览页，展示环境名称、`envId` 与主要入口位置。
2. `cloudbase-custom-login.png`
   `用户管理 -> 登录方式 -> 自定义登录` 页面入口或开启状态。
3. `cloudbase-create-collection.png`
   数据库新建集合页面。
4. `cloudbase-auth-http-entry.png`
   `modmin_auth` 云函数页面中 HTTP 访问服务入口。
5. `cloudbase-auth-http-url.png`
   开启 HTTP 访问服务后生成访问地址的位置。

命名建议：

1. 使用英文小写 + 中划线。
2. 命名与部署指南中的占位说明保持一致。
3. 优先使用 PNG，保证控制台文字清晰可辨。
